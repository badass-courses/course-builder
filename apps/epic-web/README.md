# Epic Web Self-Serve Platform

This is the Epic Web Self-Serve Platform, built on top of Course Builder.

## Features

- Content management for courses, modules, and lessons
- Section support for better content organization
- Quick editing capabilities with context menus
- Drag and drop list configuration
- AI-powered content assistance

## Getting Started

```bash
# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local

# Run the development server
pnpm dev
```

## Development

This project uses:

- Next.js with App Router
- Drizzle ORM with MySQL
- tRPC for API
- Tailwind CSS for styling
- PartyKit for real-time collaboration
